# Decisiones de diseño pendientes

## Header responsive con dropdown de usuario

**Fecha**: 14 diciembre 2025
**Estado**: Por revisar

### Propuesta
En lugar de menú hamburguesa tradicional, implementar:

**Desktop**: 
- Logo | Navegación principal (Inicio, Explorar, Mi colección) | Dropdown de usuario (avatar + menú)

**Mobile**: 
- Logo | Icono de usuario (abre dropdown)
- Opcional: Bottom navigation bar para navegación principal

### Dropdown de usuario incluiría:
- Mi perfil
- Configuración
- Favoritos/Listas
- Cerrar sesión

### Razón
Para una app de música/discos, la navegación principal es limitada (3-4 items) y puede estar siempre visible. El menú hamburguesa es innecesario. El dropdown de usuario centraliza todas las opciones de cuenta.

### Consideraciones
- ¿El dropdown de usuario es suficiente o necesitamos también navegación hamburguer?
- ¿Bottom nav en móvil o header compacto con todo?
- ¿Cómo manejamos notificaciones si las hay?
